# Yedekli_ve_Kesintisiz_Veri_Akis_Sistemi
Yedekli ve Kesintisiz Veri Akış Sistemi İHA Komuta Kontrol (C2) mimarileri için tasarlanmış, Python ve Redis tabanlı, hata toleranslı (fault-tolerant) veri yönetim simülasyonu. Docker konteynerizasyonu ile yüksek erişilebilirlik ve gerçek zamanlı veri senkronizasyonu yeteneklerini sergiler.
